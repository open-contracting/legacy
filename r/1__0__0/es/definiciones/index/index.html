The Open Contracting Data Standard is a core product of the Open Contracting Partnership (OCP).
Version 1.0 of the standard is being developed for the OCP by the World Wide Web Foundation,

Ayuda

Para poder hacer comentarios, debe iniciar sesión. Para ello, haga clic sobre el botón que aparece en la barra superior. Una vez inicie sesión, podrá seleccionar frases o párrafos del documento. Cuando lo haga aparecerá el icono que se muestra en la imagen de abajo. Haga clic sobre el icono para introducir el texto que quiera y que este sea visible.

Si tiene algún problema, no dude en enviar un correo electrónico a nuestra lista de distribución.
